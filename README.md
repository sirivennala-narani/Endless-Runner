**Welcome to Endless Runner**
This is an Expo project built as an endless runner game with jumping, obstacle dodging, score tracking, and high score saving. It uses file-based routing with Expo Router, and you can start developing by editing the files inside the app directory.

**Get started
Install dependencies**.

**bash**
npm install

Start the app.

**bash**
npx expo start

In the output, you’ll find options to open the game in a development build, Android emulator, iOS simulator, or Expo Go.

**Project structure**
You can start developing by editing the files inside the app directory, since this project uses file-based routing. The main gameplay screen contains the player cube, obstacles, collision detection, score updates, and restart logic. The game also saves the high score locally using AsyncStorage.
